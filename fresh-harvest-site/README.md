# Тепличный урожай — фермерский сайт для GitHub Pages

Готовый адаптивный сайт-лендинг для продажи огурцов, томатов, зелени и сезонных наборов.

## Что внутри

- `index.html` — вся структура сайта
- `styles.css` — стили, анимации, адаптивность
- `script.js` — каталог, фильтры, корзина, форма заказа, скролл-эффекты
- `assets/images/*` — фотографии для сайта
- `assets/favicon.svg` — иконка

## Фишки сайта

- современный hero-блок с живыми фотографиями
- курсор в виде огурчика на десктопе
- индикатор прокрутки с огурчиком
- каталог с фильтрами
- локальная корзина через `localStorage`
- форма заявки с генерацией письма
- мягкие анимации появления секций
- адаптация под мобильные устройства

## Как опубликовать на GitHub Pages

1. Создайте новый репозиторий на GitHub.
2. Загрузите содержимое этой папки в корень репозитория.
3. Откройте `Settings` → `Pages`.
4. Включите публикацию из ветки `main` и папки `/root`.
5. GitHub выдаст вам ссылку на сайт.

## Что заменить под себя

### Email для заявок
Откройте `script.js` и замените строку:

```js
const contactEmail = 'hello@fresh-harvest.local';
```

на ваш реальный email.

### Тексты, цены, товары
Все товары лежат в массиве `products` внутри `script.js`.

### Фотографии
Можно заменить изображения в папке `assets/images`, сохранив их имена или изменив пути в `index.html` и `script.js`.
